# ✅ v1.3.0 划词翻译支持语言选择

## 🎉 新功能上线！

划词翻译现在支持选择目标语言了！

---

## 🆕 本次更新

### 核心功能：划词翻译语言选择

**之前：**
- 选中文字 → 点击「🌐 翻译」→ 显示结果
- 只能翻译成默认语言

**现在：**
- 选中文字 → **选择目标语言** → 点击「翻译」→ 显示结果
- 支持 8 种语言随意切换

---

## 🎯 新界面设计

### 翻译工具栏

```
┌─────────────────────┐
│ [中文 ▼]  [翻译]    │  ← 横向布局
└─────────────────────┘
```

**组成部分：**
1. **语言选择器**（下拉框）
   - 中文、英语、日语、韩语
   - 法语、德语、西班牙语、俄语
   
2. **翻译按钮**（紫色）
   - 点击开始翻译
   - 显示结果气泡

---

## ✨ 智能语言检测

### 自动判断默认语言

选中文字后，插件会智能判断：

- **中文文字** → 默认翻译成「英语」
- **其他文字** → 默认翻译成「中文」

**示例：**
```
选中 "你好世界"
→ 默认显示 [英语 ▼] [翻译]
→ 可以改选其他语言

选中 "Hello World"
→ 默认显示 [中文 ▼] [翻译]
→ 可以改选其他语言
```

## 📋 使用步骤

### 1. 选中文字

在任何网页上选中要翻译的文字

### 2. 选择目标语言

从下拉框中选择想要翻译成的语言：
- 中文
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语
- 俄语

### 3. 点击翻译

点击「翻译」按钮，查看结果

### 4. 查看结果

翻译结果显示在气泡窗口中
