# Simple AI Translate - 简单 AI 翻译插件

【纯 cursor 实现】一个功能强大的浏览器翻译插件，支持网页划词翻译和自定义输入翻译。

## ✨ 功能特性

### 📝 划词翻译
- 在任何网页上选中文字，点击翻译按钮
- 主动可控，避免误触
- 简洁的翻译气泡显示
- 支持 ESC 键快速关闭

### 🎯 自定义翻译
- 在弹出窗口中输入任意文本进行翻译
- 支持多种语言互译
- 实时字符计数（最多 5000 字符）
- 一键复制译文
- 语言快速交换

### 🌍 支持语言
- 中文（简体）
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语
- 俄语

## 🚀 安装方法

### Chrome / Edge 浏览器

1. 打开浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启右上角的「开发者模式」

3. 点击「加载已解压的扩展程序」

4. 选择 `simple-ai-translate` 文件夹

5. 安装完成！

## 📖 使用指南

### 划词翻译
1. 在任何网页上用鼠标选中需要翻译的文字
2. 点击弹出的「🌐 翻译」按钮
3. 翻译结果显示在气泡窗口中
4. 点击 × 按钮或按 ESC 键关闭

### 弹出窗口翻译
1. 点击浏览器工具栏上的插件图标
2. 在输入框中输入要翻译的文本
3. 选择源语言和目标语言
4. 点击「✨ 翻译」按钮（或按 Ctrl/Cmd + Enter）
5. 使用「📋 复制」按钮复制译文

### 快捷操作
- **点击翻译按钮**: 触发划词翻译
- **Ctrl/Cmd + Enter**: 在弹出窗口快速翻译
- **ESC**: 关闭翻译气泡
- **⇄ 按钮**: 交换源语言和目标语言

## 🌐 翻译 API

本插件使用以下免费翻译 API（按优先级）：

1. **LibreTranslate** - 开源免费翻译服务
2. **MyMemory** - 免费翻译 API
3. 可扩展支持其他翻译服务

## 📝 注意事项

1. 翻译功能需要网络连接
2. 免费 API 可能有调用频率限制
3. 翻译质量取决于所使用的 API 服务
4. 语音朗读功能依赖浏览器的 TTS 引擎
