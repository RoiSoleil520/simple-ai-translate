## 🚀 如何使用新版本

### 更新插件

1. 打开 Chrome：`chrome://extensions/`
2. 找到「Simple AI Translate」
3. 点击刷新按钮 🔄
4. 完成！

### 新的使用方式

**划词翻译：**
```
1. 选中文字
   ↓
2. 点击「🌐 翻译」按钮
   ↓
3. 查看翻译结果
```

**弹出窗口：**
- 保持原有方式不变
- 点击插件图标 → 输入文本 → 点击翻译
