# 图标生成说明

## 方法 1：使用图标生成器（推荐）

1. 在浏览器中打开 `icon-generator.html` 文件
2. 点击对应的按钮下载三个尺寸的图标：
   - icon16.png (16x16)
   - icon48.png (48x48)
   - icon128.png (128x128)
3. 将下载的图标放在 `icons` 文件夹中

## 方法 2：使用在线工具

你也可以使用在线图标生成工具：

1. 访问 https://www.favicon-generator.org/ 或类似网站
2. 上传你的设计或使用文字生成图标
3. 下载不同尺寸的图标
4. 重命名为 icon16.png、icon48.png、icon128.png
5. 放在 icons 文件夹中

## 方法 3：使用现有图标

如果你想快速测试，可以从其他项目复制图标文件，或者使用简单的图片编辑器创建：

- 背景色：渐变色 #667eea 到 #764ba2
- 图案：地球或翻译相关的图标
- 字母 "A" 表示 AI 翻译

## 临时方案

如果你暂时没有图标，浏览器会显示默认的扩展图标，不影响插件功能的正常使用。
