# 🚀 快速开始

## 三步即可使用

### 1️⃣ 生成图标（30 秒）

在浏览器中打开：

```
simple-ai-translate/icons/icon-generator.html
```

点击红色按钮「一键下载全部图标」，将下载的 3 个 PNG 文件保存到 `icons` 文件夹。

### 2️⃣ 安装插件（10 秒）

**Chrome / Edge：**
1. 地址栏输入：`chrome://extensions/` 或 `edge://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `simple-ai-translate` 文件夹

### 3️⃣ 开始使用（立即！）

✅ **划词翻译**：在任意网页选中文字，即刻翻译  
✅ **输入翻译**：点击工具栏图标，输入内容翻译
