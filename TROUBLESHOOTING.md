# 🔧 故障排除指南

## 常见问题和解决方案

### ❌ 翻译报错："'AUTO' IS AN INVALID SOURCE LANGUAGE"

**问题原因：**
- 旧版本的插件使用的 MyMemory API 不支持自动检测语言

**解决方案：**
1. 确保你使用的是最新版本（v1.0.1 或更高）
2. 重新加载插件：
   - 打开 `chrome://extensions/`
   - 找到「Simple AI Translate」
   - 点击刷新按钮 🔄

**已修复！** v1.0.1 版本添加了智能语言检测功能。

---

### ❌ 翻译功能完全不工作

**可能原因和解决方案：**

#### 1. 网络问题
- ✅ 检查网络连接
- ✅ 尝试访问 https://translate.googleapis.com
- ✅ 检查防火墙设置

#### 2. 插件未正确安装
```bash
步骤：
1. 打开 chrome://extensions/
2. 确保「Simple AI Translate」已启用
3. 查看是否有错误提示
4. 尝试重新加载插件
```

#### 3. 浏览器控制台错误
```bash
查看错误信息：
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查看红色错误信息
4. 根据错误信息排查问题
```

---

### ❌ 划词翻译气泡不显示

**检查清单：**

- [ ] 是否选中了足够的文字（至少 1 个字符）
- [ ] 是否在特殊页面（如 chrome:// 页面不允许运行扩展）
- [ ] 刷新网页后重试
- [ ] 检查是否有其他扩展冲突

**解决步骤：**
1. 刷新网页（Ctrl+R / Cmd+R）
2. 重新选中文字
3. 打开控制台（F12）查看错误
4. 重启浏览器

---

### ❌ 弹出窗口翻译无反应

**检查项目：**

1. **输入框是否有内容**
   - 确保输入了要翻译的文字
   - 检查字符数是否超过 5000

2. **目标语言是否与源语言相同**
   - 例如：中文翻译成中文会没有效果
   - 尝试更换目标语言

3. **查看浏览器控制台**
   - 右键插件图标 → 检查弹出窗口
   - 查看 Console 中的错误信息

---

### ❌ 复制功能不工作

**解决方案：**

1. **浏览器权限问题**
   - 现代浏览器需要用户交互才能复制
   - 确保点击了复制按钮

2. **剪贴板权限**
   - Chrome 可能需要额外的剪贴板权限
   - 检查 `chrome://settings/content/clipboard`

3. **手动复制**
   - 如果自动复制失败
   - 可以手动选择文字并复制（Ctrl+C / Cmd+C）

---

### ❌ 语音朗读没有声音

**可能原因：**

1. **浏览器不支持 TTS（文本转语音）**
   - Chrome/Edge 默认支持
   - 某些浏览器可能需要安装语音包

2. **系统音量设置**
   - 检查系统音量是否静音
   - 检查浏览器是否被静音

3. **语言包缺失**
   - Windows: 设置 → 时间和语言 → 语音
   - Mac: 系统偏好设置 → 辅助功能 → 语音
   - 安装对应语言的语音包

---

### ❌ 翻译速度很慢

**优化建议：**

1. **网络问题**
   - 使用稳定的网络连接
   - 某些 API 可能被限速

2. **文本过长**
   - 尽量控制在 1000 字符以内
   - 长文本可以分段翻译

3. **API 切换**
   - 插件会自动尝试多个 API
   - 如果第一个 API 慢，会自动切换

---

### ❌ 特定语言翻译不准确

**说明：**

翻译质量取决于使用的 API 服务：
- Google Translate：通常最准确
- LibreTranslate：开源，质量中等
- MyMemory：免费但可能不太准确

**改进建议：**
1. 使用更精确的源语言（而不是"自动检测"）
2. 检查原文是否有拼写错误
3. 尝试重新翻译
4. 对于专业术语，建议使用专业翻译工具

---

### 🔍 调试模式

如果以上方法都无法解决问题，启用调试模式：

#### 1. 打开背景页调试
```bash
1. 打开 chrome://extensions/
2. 找到「Simple AI Translate」
3. 点击「检查视图」→「Service Worker」
4. 查看 Console 中的日志
```

#### 2. 打开内容脚本调试
```bash
1. 在任意网页按 F12
2. 切换到 Console 标签
3. 选中文字触发翻译
4. 查看日志信息（以 "Simple AI Translate:" 开头）
```

#### 3. 查看网络请求
```bash
1. 按 F12 打开开发者工具
2. 切换到 Network 标签
3. 触发翻译
4. 查看 API 请求是否成功（200 状态码）
```

### 📞 仍然无法解决？

如果以上方法都无法解决你的问题：

1. **收集信息：**
   - 浏览器版本
   - 插件版本
   - 错误信息截图
   - Console 日志

2. **检查更新：**
   - 查看是否有新版本
   - 阅读 CHANGELOG.md 了解最新修复

3. **重新安装：**
   - 完全卸载插件
   - 删除缓存数据
   - 重新安装最新版本
