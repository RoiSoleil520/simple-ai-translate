# 📦 安装指南

## 第一步：生成图标

在安装插件之前，你需要先生成图标文件。选择以下任一方法：

### 方法 A：使用 HTML 生成器（最简单）✨

1. 用浏览器打开 `icons/icon-generator.html` 文件
2. 点击三个按钮，依次下载：
   - icon16.png
   - icon48.png
   - icon128.png
3. 将下载的文件放在 `icons` 文件夹中

### 方法 B：使用 Python 脚本

```bash
# 安装 Pillow 库（如果还没安装）
pip install Pillow

# 进入 icons 文件夹并运行脚本
cd icons
python3 generate_icons.py
```

### 方法 C：跳过图标（临时方案）

如果你想快速测试，可以先跳过图标生成。插件会显示默认图标，但功能完全正常。

## 第二步：安装插件到浏览器

### Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `simple-ai-translate` 文件夹
6. 完成！你会在工具栏看到插件图标

### Edge 浏览器

1. 打开 Edge 浏览器
2. 在地址栏输入：`edge://extensions/`
3. 开启左下角的「开发人员模式」
4. 点击「加载解压缩的扩展」
5. 选择 `simple-ai-translate` 文件夹
6. 完成！

## 第三步：开始使用

### 🎯 划词翻译

1. 打开任意网页
2. 用鼠标选中任意文字
3. 翻译结果会自动弹出显示

### ✍️ 自定义输入翻译

1. 点击浏览器工具栏的插件图标
2. 在输入框中输入要翻译的内容
3. 选择源语言和目标语言
4. 点击「翻译」按钮（或按 Ctrl/Cmd + Enter）

## 🎨 功能特性

- ✅ 网页划词即时翻译
- ✅ 支持 8 种常用语言
- ✅ 一键复制译文
- ✅ 语音朗读功能
- ✅ 语言快速交换
- ✅ 优雅的现代化界面
- ✅ 字符计数（最多 5000 字符）
- ✅ 快捷键支持（Ctrl/Cmd + Enter 翻译，ESC 关闭）

## ⚠️ 注意事项

1. **网络连接**：翻译功能需要网络连接
2. **API 限制**：使用免费翻译 API，可能有调用频率限制
3. **隐私**：所有翻译请求都是直接发送到第三方翻译服务，本插件不存储任何数据

## 🔧 故障排除

### 问题：插件图标不显示

**解决方案**：
- 确保 `icons` 文件夹中有 icon16.png、icon48.png、icon128.png 三个文件
- 如果没有，按照「第一步」生成图标
- 重新加载插件

### 问题：翻译功能不工作

**解决方案**：
1. 检查网络连接
2. 打开浏览器控制台（F12）查看错误信息
3. 尝试刷新页面
4. 重新加载插件

### 问题：划词翻译没有反应

**解决方案**：
1. 刷新网页
2. 检查是否选中了足够的文字（至少 1 个字符）
3. 确保插件已正确安装并启用

## 💡 使用技巧

1. **快速翻译**：在弹出窗口按 Ctrl/Cmd + Enter 快速翻译
2. **关闭气泡**：按 ESC 键关闭划词翻译气泡
3. **交换语言**：点击中间的 ⇄ 按钮快速交换源语言和目标语言
4. **复制译文**：点击「复制」按钮或手动选择复制
5. **语音朗读**：点击「朗读」按钮听发音（需要浏览器支持）

## 📞 需要帮助？

如果遇到问题，请检查：
1. 浏览器控制台的错误信息（按 F12 打开）
2. 插件是否正确安装并启用
3. 网络连接是否正常
